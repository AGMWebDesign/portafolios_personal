Sobre mí

Soy un desarrollador web junior que ha aprendido de forma autodidacta, impulsado por la curiosidad y las ganas de construir cosas desde cero.

Trabajo con HTML, CSS y JavaScript para crear sitios web que no solo se ven bien, sino que también funcionan de manera eficiente en diferentes dispositivos.

Aunque estoy comenzando en este camino, me esfuerzo por mejorar constantemente, seguir aprendiendo y ofrecer resultados que generen valor real a quienes confían en mi trabajo.

Servicios

Ofrezco servicios de desarrollo web centrados en crear sitios modernos, responsivos y funcionales utilizando HTML, CSS y JavaScript. Trabajo con un enfoque en la simplicidad, claridad y adaptabilidad, pero también puedo implementar funciones más avanzadas si tu proyecto lo requiere.

  • Creación de páginas web responsivas: Sitios adaptados a todo tipo de dispositivos, con un diseño fluido y optimizado.
  • Diseño web personalizado: Desarrollo basado en tus ideas, marca o estilo, para lograr un sitio único y alineado con tus objetivos.
  • Mantenimiento y actualizaciones: Ajustes, correcciones y mejoras continuas para mantener tu sitio siempre al día.
